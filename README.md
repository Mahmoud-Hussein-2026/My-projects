# Scientific Calculator
A Java project for a Scientific Calculator with multiple operations.
## 🚀 Features
- Basic Math: +, -, *, /, mod.
- Advanced Math: pow, !, sq, cb.
- Functions: sin, cos, tan, ln.
- Others: max, min.
