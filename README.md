# 🏥 Hospital Management System

A console-based application built using **Object-Oriented Programming (OOP)** to manage healthcare operations, tracking doctors, patients, and staff allocations within a hospital environment.

## 🛠️ Features & Implementation:
* **Healthcare Management:** Register and track records for both medical staff (**Doctors**) and consumers (**Patients**) seamlessly.
* **OOP Concepts:** Built on top of robust **Inheritance** hierarchy where `Doctor` and `Patient` classes cleanly inherit core fields from the base `Person` class.
* **Polymorphic Design:** Extensive usage of Method Overriding via customized `@Override toString()` methods to produce structured data logs for entities.
* **Data Management:** Dynamic data structures using **`ArrayList`** collections inside the `Hospital` class to handle system growth in-memory.
* **User Interactivity:** Console-driven menu routing inside `MainHospital` that efficiently matches input commands to precise system subroutines.
* **Source Code:** [Person.java](./Person.java), [Doctor.java](./Doctor.java), [Patient.java](./Patient.java), [Hospital.java](./Hospital.java), [MainHospital.java](./MainHospital.java)
